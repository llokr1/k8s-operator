# 쿠버네티스 오퍼레이터 학습 및 구현

## 배경

---

프로젝트를 통해 쿠버네티스 환경에서 서비스를 개발하며 컨테이너 오케스트레이션 도구로서의 쿠버네티스를 경험할 수 있었다. 그러나 이후 인턴십을 통해 실제 운영 환경의 쿠버네티스를 접하면서, 단순히 컨테이너를 배포하고 관리하는 수준을 넘어 다양한 리소스를 자동으로 운영하고 관리하는 ‘운영 자동화 도구’로 활용되고 있음을 깨닫게 되었다.

특히, 특정 플랫폼의 여러 구성 요소들이 쿠버네티스 상에서 오퍼레이터 패턴을 기반으로 관리되는 것을 보면서, 쿠버네티스가 단순한 오케스트레이션 도구를 넘어, 운영 자동화를 구현하는 핵심 플랫폼으로 활용될 수 있음을 이해하게 되었다.

이에 따라 쿠버네티스의 활용도를 더욱 높이고, 실제 운영 환경에서의 자동화 메커니즘을 깊이 있게 이해하기 위해 오퍼레이터 패턴에 대한 학습의 필요성을 느끼게 되었고, 이를 본격적으로 공부하게 되었다.
